# Family Tree Creator

A starter scaffold for a family tree builder web app.

## Project structure

- `public/`
  - `index.html`
  - `favicon.ico`
  - `assets/`
    - `icons/`
    - `images/`
- `src/`
  - `css/`
  - `js/`
    - `data/`
    - `models/`
    - `services/`
    - `ui/`
    - `utils/`
  - `templates/`
- `tests/`
- `.gitignore`
- `package.json`
- `vite.config.js`

## Getting started

1. Install dependencies: `npm install`
2. Run local development server: `npm run dev`
3. Run tests: `npm test`
